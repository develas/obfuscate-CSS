# Obfuscator-CSS

**Allows you to obfuscate CSS classes.**  
**Usege:**  
The Obfuscator module accepts array with your CSS classes and returns object with new CSS classes.  
['class1', 'class1', 'class2'] -> {class1: 'a', class2: 'b'}

**This module also makes minification of classes.**  
So, the most common classes will have the shortest name.  

**Run tests:**  
**npm test**
